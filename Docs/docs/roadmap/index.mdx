## Next milestone

**Unlit ECS Scene Renderer** — the shortest path from "draw one hard-coded model" to "render an ECS world":

1. Add `SafiGlobalTransform` and a transform propagation system
2. Add an engine-owned render system that queries `(SafiTransform, SafiMeshRenderer)`
3. Add an active camera concept
4. Render every matching entity with the current unlit material path

This unlocks visibility, multi-camera, scene loading, asset handles, and lit materials without requiring PBR or physics first.

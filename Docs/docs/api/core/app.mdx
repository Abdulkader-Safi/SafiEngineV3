# `SafiApp`

`SafiApp` owns the window, the SDL_gpu renderer, the flecs world, and the main loop. Every SafiEngine program creates exactly one.

```c
#include <safi/core/app.h>
```

## Types

```c
typedef struct SafiAppDesc {
    const char *title;
    int         width;
    int         height;
    bool        vsync;
    bool        enable_debug_ui;
    /* Seconds per fixed-update step. 0 ‚Üí default (1/60). */
    float       fixed_dt;
    /* Max fixed steps per frame (spiral-of-death cap). 0 ‚Üí default (4). */
    int         fixed_max_steps;
} SafiAppDesc;

typedef struct SafiApp {
    SafiRenderer  renderer;
    ecs_world_t  *world;
    bool          running;
    // ...internal fields...
    float         fixed_dt;           // seconds per fixed step
    float         fixed_accumulator;  // unconsumed dt
    float         fixed_elapsed;      // accumulated fixed time
    int           fixed_max_steps;    // spiral-of-death cap
} SafiApp;
```

## Functions

### `safi_app_init`

```c
bool safi_app_init(SafiApp *app, const SafiAppDesc *desc);
```

Creates the window, initialises SDL_gpu, builds a flecs world pre-populated with the [stock components](../ecs/components), the three engine pipelines, and the engine-owned [render system](../render/overview) on `EcsOnStore`. Installs the `SafiInput` and `SafiTime` singletons, and optionally boots the MicroUI overlay. Returns `false` on failure; check `SDL_GetError()` for details.

### `safi_app_run`

```c
void safi_app_run(SafiApp *app);
```

Blocks the calling thread in the main loop until the window closes or a system sets `app->running = false`. Each frame, `safi_app_tick` drives three pipelines in order:

1. **Variable** ‚Äî `EcsOnLoad ‚Üí ‚Ä¶ ‚Üí EcsPostUpdate` at wall-clock `dt`
2. **Fixed** ‚Äî `SafiFixedUpdate`, looped N times at stable `fixed_delta`
3. **Render** ‚Äî `EcsPreStore ‚Üí EcsOnStore` at wall-clock `dt`

See [Scheduler & Stages](./scheduler) for the details.

### `safi_app_tick`

```c
bool safi_app_tick(SafiApp *app);
```

Runs a single frame. Useful when embedding the engine in another loop or writing tests. Returns `false` when the app should exit.

### `safi_app_shutdown`

```c
void safi_app_shutdown(SafiApp *app);
```

Tears everything down in reverse order (ImGui ‚Üí world ‚Üí renderer ‚Üí SDL).

### `safi_app_world`

```c
static inline ecs_world_t *safi_app_world(SafiApp *app);
```

Returns the underlying flecs world so you can register systems and spawn entities.

## Example

```c title="examples/gltf_viewer/main.c"
SafiApp app;
safi_app_init(&app, &(SafiAppDesc){
    .title = "SafiEngine ‚Äî glTF Viewer",
    .width = 1280, .height = 720,
    .vsync = true,
    .enable_debug_ui = true,
});

ecs_world_t *world = safi_app_world(&app);
// ...register systems, spawn entities...

safi_app_run(&app);
safi_app_shutdown(&app);
```
